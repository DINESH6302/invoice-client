# BizBill Manager

A modern billing application for managing business invoices.

## Features
- Dynamic Invoice Template Builder (3-pane design)
- Custom fields support
- Next.js + Tailwind CSS + Shadcn UI (Architecture)

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the development server:
   ```bash
   npm run dev
   ```

3. Open [http://localhost:3000](http://localhost:3000) with your browser to see the Invoice Template Builder.
